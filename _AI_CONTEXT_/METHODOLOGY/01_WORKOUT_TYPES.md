## Overview

This library defines all workout types recognized by the system. Each workout serves a specific physiological purpose. The classification enables:
- Automatic tagging of activities
- Training load distribution analysis
- Performance tracking by workout type

---

## Recovery & Easy Runs

### Recovery Run
- **Duration:** 20-45 min
- **Intensity:** <75% MHR, very easy
- **Pace:** 1:30-2:00+/mi slower than race pace
- **Purpose:** Active recovery, blood flow without stress
- **When:** Day after hard workout or race

### Easy Run
- **Duration:** 30-90 min
- **Intensity:** 72-81% MHR
- **Pace:** Conversational, full sentences
- **Purpose:** Aerobic base, recovery
- **When:** Majority of weekly mileage

### Shake-out Run
- **Duration:** 15-30 min
- **Intensity:** <70% MHR
- **Purpose:** Pre-race activation, loosen legs
- **When:** Day before or morning of race

---

## Endurance Runs

### Long Run (Easy)
- **Duration:** 1-3+ hours
- **Intensity:** 75-83% MHR
- **Pace:** 10-20% slower than marathon pace
- **Purpose:** Endurance, fat oxidation, mental toughness
- **When:** Weekly, typically weekend

### Medium-Long Run
- **Duration:** 70-75% of long run distance
- **Intensity:** Same as long run
- **Purpose:** Build endurance without full recovery cost
- **When:** Mid-week endurance builder

### Aerobic Run
- **Duration:** 40-70 min
- **Intensity:** 72-81% MHR
- **Purpose:** Steady aerobic development
- **When:** Regular training days

## Threshold / Stamina Work

### Tempo Run (Continuous)
- **Duration:** 10-40 min at tempo
- **Intensity:** 83-90% MHR, "comfortably hard"
- **Pace:** ~10-15 sec/mi slower than 10K pace
- **Purpose:** Improve lactate threshold
- **When:** 1-2x per week during build phase

### Tempo Intervals
- **Duration:** 5-20 min per interval
- **Recovery:** 1-2 min jog
- **Intensity:** Same as tempo
- **Purpose:** Threshold work with recovery breaks
- **When:** Alternative to continuous tempo

### Cruise Intervals
- **Duration:** 3-8 min per interval
- **Recovery:** 30-60 sec jog
- **Intensity:** Threshold pace
- **Purpose:** Threshold with rhythm focus
- **When:** Build toward continuous tempo

## Speed / VO2max Work

### VO2max Intervals
- **Duration:** 3-5 min per rep
- **Recovery:** Equal to or slightly less than work time
- **Intensity:** 5K race pace, 93-95% MHR
- **Purpose:** Increase aerobic power
- **Volume limit:** Max 8% of weekly mileage OR 10K total

### 5K Pace Intervals
- **Duration:** 800m-1600m reps
- **Recovery:** 50-90% of work time
- **Intensity:** 5K race effort
- **Purpose:** Race-specific speed
- **When:** Peak phase for 5K-10K

### 10K Pace Intervals
- **Duration:** 1000m-2000m reps
- **Recovery:** 50-75% of work time
- **Intensity:** 10K race effort
- **Purpose:** Race-specific endurance speed
- **When:** Peak phase for 10K-half

### Fartlek
- **Duration:** Variable (30 sec - 5 min bursts)
- **Intensity:** 5K-10K effort for fast portions
- **Recovery:** Easy jog between
- **Purpose:** Speed + endurance, flexible
- **When:** Anytime, good for mental variety

## Repetitions & Strides

### Repetitions (Short)
- **Duration:** 200m-600m (under 2 min)
- **Recovery:** 2-3x work duration
- **Intensity:** Faster than 5K pace
- **Purpose:** Speed, economy, anaerobic power
- **Volume limit:** Max 5% of weekly mileage OR 5 miles

### Strides
- **Distance:** 80-120m
- **Intensity:** Fast but relaxed (not sprinting)
- **Recovery:** Full (walk back)
- **Purpose:** Running economy, neuromuscular activation
- **When:** After easy runs, 4-6 reps

### Hill Sprints
- **Duration:** 8-10 seconds
- **Intensity:** Maximum effort
- **Recovery:** Full (2-3 min)
- **Grade:** 6-10% incline
- **Purpose:** Neuromuscular power, economy
- **When:** After easy runs, 6-12 reps

### Hill Repetitions
- **Duration:** 2-5 min uphill
- **Intensity:** 5K effort
- **Recovery:** Jog down
- **Purpose:** Leg strength, power endurance
- **When:** Base/build phase

## Race-Specific Work

### Marathon Pace Run
- **Duration:** 8-16 miles at MP
- **Intensity:** Goal marathon pace, 81-88% MHR
- **Purpose:** Race-specific fitness, confidence
- **When:** Build and peak phase

### Half-Marathon Pace Run
- **Duration:** 6-10 miles at HMP
- **Intensity:** Goal half pace, 83-88% MHR
- **Purpose:** Race-specific fitness
- **When:** Build and peak phase

### Goal Pace Intervals
- **Duration:** 1-3 miles per rep
- **Intensity:** Exact target race pace
- **Purpose:** Pace familiarity, confidence
- **When:** Race-specific phase

### Race Simulation
- **Duration:** 60-75% of race distance
- **Intensity:** Goal race pace
- **Purpose:** Full dress rehearsal
- **When:** 2-3 weeks before goal race

### Tune-Up Race
- **Duration:** Race distance (shorter than goal)
- **Intensity:** Race effort
- **Purpose:** Fitness test, race practice
- **When:** During build phase

## Progression Runs

### Basic Progression
- **Structure:** Start slow, finish faster
- **Duration:** Any length
- **Purpose:** Teaches pacing, builds finishing strength

### Thirds Run
- **Structure:** 1/3 slow → 1/3 steady → 1/3 fast
- **Duration:** Typically 6-10 miles
- **Purpose:** Progressive effort, controlled build

### Fast Finish Run
- **Structure:** 75-90% easy, then faster final portion
- **Duration:** Any length
- **Purpose:** Finishing strength, mental toughness

### Super Fast Finish
- **Structure:** Easy run + sprint last 2-3 min
- **Duration:** Any length
- **Purpose:** Neuromuscular activation, kick practice

---

## Long Run Variants

### Easy Long Run
- **Pace:** 10-20% slower than marathon pace
- **Purpose:** Pure endurance building
- **When:** Base phase, recovery weeks

### Progressive Long Run
- **Structure:** Start easy, each segment slightly faster
- **Final pace:** Near marathon pace
- **Purpose:** Teaches negative splitting under fatigue

### Fast Finish Long Run
- **Structure:** Easy → race pace finish (last 20-30%)
- **Purpose:** Simulate race fatigue, mental toughness
- **Example:** 18 mi: 14 mi easy, 4 mi @ MP

### Marathon Pace Long Run
- **Duration:** Extended miles at marathon pace
- **Purpose:** Goal pace confidence, race simulation
- **When:** Peak phase, 3-4 weeks out

### Cutdown Long Run
- **Structure:** Each 3-4 mile segment faster than previous
- **Purpose:** Extreme pacing practice, mental toughness

### Specific-Endurance Long Run
- **Structure:** Easy + race pace segments interspersed + easy
- **Example:** 1 hr easy, 15 × 1 min @ MP / 1 min easy, 30 min easy
- **Purpose:** Race simulation with controlled fatigue

## Auto-Classification Rules

### By Heart Rate Zone
| Avg HR | Classification |
|--------|----------------|
| <75% MHR | Recovery |
| 72-83% MHR (steady) | Easy/Long |
| 81-88% MHR (steady) | Marathon Pace |
| 83-90% MHR | Threshold/Tempo |
| 93-95% MHR | VO2max/Intervals |
| >95% MHR | Repetitions |

### By Pace Variance
| Pattern | Classification |
|---------|----------------|
| Steady pace throughout | Easy/Tempo/MP |
| Intervals detected (fast/slow) | Interval workout |
| Progressive (each segment faster) | Progression run |
| Fast finish pattern | Fast Finish variant |

### By Duration
| Duration | If Easy Pace |
|----------|--------------|
| <30 min | Recovery/Shake-out |
| 30-60 min | Easy Run |
| 60-90 min | Aerobic/Medium-Long |
| >90 min | Long Run |

## Workout Distribution Guidelines

### Weekly Structure (Example)
| Day | Workout Type |
|-----|--------------|
| Mon | Recovery or Off |
| Tue | Quality (Intervals/Tempo) |
| Wed | Easy |
| Thu | Quality or Steady |
| Fri | Easy or Recovery |
| Sat | Long Run |
| Sun | Easy or Off |

### Intensity Distribution
- **80%+ Easy/Recovery:** Builds aerobic base, allows adaptation
- **<20% Quality:** Sufficient stimulus without overreaching

### Volume Limits for Quality
- **VO2max work:** Max 8% of weekly mileage
- **Repetition work:** Max 5% of weekly mileage
- **Threshold work:** Max 10-15% of weekly mileage
