## Key principles extracted (resource summary)

1. 1500/mile performance is framed as **speed + fatigue resistance** (speed endurance under stress).
2. Training emphasis is **quality-first**, with aerobic volume present but often secondary.
3. Uses long-cycle periodization (rest -> transition -> preparation -> pre-comp -> competition -> peak).
4. Uses multi-pace development (roughly 5k/3k/1500/800/400 pace exposures).
5. Intensive interval work increases as racing phase approaches.
6. Technical components are emphasized (acceleration mechanics, curve running, sprint economy).
7. Circuit-based strength work is treated as a core pillar, not optional add-on.
8. Workload is treated in "units" (session load) rather than strict mileage targets alone.
9. Approach is historically successful but tightly tied to athlete specificity.
