# SWAP Workout Execution Guide — Reference Note

**Source:** SWAP Running Patreon (16 articles consolidated)
**Filed:** April 10, 2026
**Relevance:** Builder execution reference for every workout type the
generator prescribes. Use this when writing workout description text,
validating generator output, or tuning the coach model's workout advice.

---

## 1. Flat Strides

**Definition:** 15-30 second accelerations on flat or gentle terrain at
fast but controlled effort (~mile to 3K effort for the last few seconds).

**Execution protocol:**
1. Accelerate smoothly over 5s — don't explode off the line
2. Build to fast but controlled (not sprinting) by 10s
3. Hold for 5-15 more seconds
4. Decelerate naturally — don't brake
5. Walk or jog 45-90s recovery between reps

**Form cues:**
- "Light on your feet, quick turnover"
- Slight forward lean from ankles, not waist
- Arms relaxed, driving forward and back
- "Imagine running on hot coals"

**When:** End of easy runs, 2-4x/week. 4-8 strides per session.
Never before the body is warm (minimum 15-20 min into a run).

**Purpose:** Running economy improvement through neuromuscular
recruitment at fast speeds. "The most efficient training intervention
per unit of time and stress."

**3-week onramp for new athletes:**
- Week 1: 4×15s at moderate effort (not truly fast)
- Week 2: 4×20s at moderate-to-fast effort
- Week 3: 6×20s at fast effort
- Week 4+: 4-8×20-30s at full stride effort

**When NOT to do strides:**
- If any musculoskeletal niggle exists
- On days with scheduled quality work (unless explicitly prescribed)
- If the athlete skipped sleep or is systemically fatigued

## 2. Hill Strides (30-45 seconds)

**Definition:** Short, powerful accelerations on steep terrain (6-10%
grade) at 3K-to-mile effort.

**Execution protocol:**
1. Start at easy pace on the hill
2. Accelerate to hard effort over 5s
3. Hold hard effort 25-40s more
4. Walk or jog down recovery (full recovery, ~90s)
5. Repeat

**Key differences from flat strides:**
- Longer duration (30-45s vs 15-30s for flat)
- Higher grade (6-10% vs flat/gentle)
- Greater muscular stress — more eccentric load on descent recovery
- More VO2 demand — these are aerobically harder

**Placement:** End of easy runs (Sun/Tue/Thu in many SWAP plans).
4-8 reps. "Hills at the end of easy runs are a staple across all
SWAP plans."

**Progression:** Start with 4×30s, build to 8×45s over 3-4 weeks.

## 3. The 200s Framework

**Definition:** A mental model for calibrating stride intensity using
200m track segments.

**How it works:**
- Run 200m on a track at what feels like "fast but I could do many more"
- Note the time — this becomes your stride reference pace
- For most athletes: ~35-45s per 200m (~mile to 3K effort)
- "If you're doing 200s in 28s, you're too fast. If 55s, too slow."

**Application to strides:**
- Each stride should feel like "a 200" regardless of terrain
- Uphill strides: same effort, slower ground speed, same neuromuscular
  fire rate
- Flat strides: same effort, faster ground speed

**Application to fast intervals (>30s):**
- 400s at "200 effort + 5s" (i.e., slightly slower per 200)
- 600s at "200 effort + 8s"
- 800s at "200 effort + 10s"
- These are effort anchors, not pace prescriptions

## 4. Hill Intervals (1-4 minutes)

**Definition:** Sustained hard efforts on moderate-to-steep terrain.
Distinct from short hill strides in duration and purpose.

### Grade Guide

| Grade | Best for | Duration |
|-------|----------|----------|
| 4-6% | Economy intervals, longer reps | 2-4 min |
| 6-8% | Power + VO2 development | 90s-3 min |
| 8-12% | Short power bursts | 45s-2 min |
| >12% | Hiking power only | N/A (walk) |

### Types

**Short hills (30-90s):**
- 8-12 reps, 3K-mile effort
- Walk/jog down recovery
- Builds power and running economy
- "The combo workout" often pairs these with steady running:
  8-12×30-45s hills + 1-3 mi steady (50K effort) afterward

**Medium hills (2-4 min):**
- 4-8 reps, 10K-to-threshold effort
- Jog down + 30s standing recovery
- Builds VO2 and lactate clearance
- Less biomechanical stress than flat intervals at same VO2 cost

**Long hills (5+ min):**
- 2-4 reps, threshold effort
- Jog down + 1min standing recovery
- Essentially threshold intervals on terrain
- Reserved for ultra athletes or specific prep

### Periodization

- Early in plan: short hills (power, economy)
- Mid plan: medium hills (VO2, lactate)
- Late plan: short hills return for race-prep sharpening
- "Hills at end of long runs" = fatigue resistance variant (different
  from standalone hill workout)

## 5. Uphill Treadmill Threshold 101

**Definition:** Sustained threshold-effort intervals on a treadmill set
to 3-6% grade. The cornerstone SWAP quality session for ultra and
marathon athletes.

### Setup

- Grade: 3-5% (4% is the sweet spot)
- Speed: whatever produces threshold effort (1-hour effort)
- "Most athletes run 1-2 min/mi slower than flat threshold on the TM"
- Heart rate should stabilize in threshold zone after 2-3 min

### Execution Protocol

1. Warm up 15-20 min easy on TM at 2-3% grade
2. Set to workout grade and speed
3. Run interval
4. Recovery: reduce speed to easy walk/jog, keep grade at 2-3%
5. Repeat

### Standard Progressions (from all SWAP plans)

| Progression | Interval | Recovery | Total work |
|-------------|----------|----------|------------|
| Entry | 6-8×3min | 2min | 18-24 min |
| Building | 6-8×4min | 90s | 24-32 min |
| Strong | 4-6×5min | 90s | 20-30 min |
| Advanced | 3-4×8min | 2min | 24-32 min |
| Peak | 2-4×10min | 2min | 20-40 min |
| Champion | 8-12×5min | 1min | 40-60 min |

**Recovery shrinks as fitness builds.** W1: 2min → W12: 1min at same
effort. This is an extension-based progression: same effort, longer
work segments, shorter recovery.

### Key Rules

- **Never chase pace on the TM.** Set it and forget it. Adjust by feel.
- **Grade over speed.** If in doubt, raise grade 0.5% rather than
  adding speed.
- **Fueling:** 30-60g carbs before or during TM sessions >60 min total.
- **Sodium bicarbonate:** "Works for some athletes. Try it in training
  first."
- **Purpose:** "The uphill TM is the single most important workout for
  ultra athletes. It provides massive aerobic stress with low impact."
- **Why uphill?** Same VO2 cost as flat threshold but:
  - Lower ground reaction forces (less injury risk)
  - Climbing-specific muscular adaptation
  - Eccentric load removed (TM belt assists)
  - Controlled, repeatable environment for threshold calibration

### Z2 Uphill Treadmill (Recovery/Supplemental)

**Definition:** Continuous easy running on TM at 4-8% grade for 30-60
minutes. NOT a quality session — a recovery or supplemental aerobic day.

- Grade: 4-8% (higher grade = slower speed = more hiking-specific)
- Speed: whatever keeps HR in Zone 2
- Duration: 30-60 min
- Purpose: recovery-day option that accumulates climbing-specific
  adaptation without impact stress
- "Better than a rest day for most athletes, better than a flat easy
  run for mountainous-race athletes"
- Can be used as the PM double on workout days
- Form benefit: forces upright posture, short stride, high cadence

## 6. Easy/Moderate Runs

**Definition:** The bread and butter. 80%+ of all running volume.

### The 5 Guidelines

1. **Start easy, end wherever your body takes you.** Not a rigid pace.
   "Easy/mod" means start in easy zone, drift to moderate if the body
   says yes. It's a brain-body negotiation.
2. **Never look at pace during the run.** Pace is reviewed after, not
   during. "Watch-glancing is the enemy of intuitive running."
3. **Light on your feet, quick turnover.** Even at slow paces, maintain
   good form. Don't shuffle.
4. **If your breathing isn't conversational in the first 20 min, you're
   too fast.** The conversational test applies at the start, not the end.
5. **Progression is natural, not forced.** On good days, the second half
   naturally gets faster. On bad days, the whole run stays easy. Both
   are correct.

### What "Easy" Actually Means (physiologically)

- Below ventilatory threshold 1 (VT1)
- Fat oxidation dominant
- Conversational
- HR roughly 60-75% max (individual-dependent)
- RPE 3-5 out of 10

### What "Moderate" Means

- Between VT1 and VT2
- Mixed fuel source
- Can speak in sentences but not paragraphs
- HR roughly 75-85% max
- RPE 5-7 out of 10
- "Moderate" in SWAP is NOT threshold. It's a comfortable-but-purposeful
  effort. "Running with intention."

### Distance Ranges

All easy/mod runs are prescribed as ranges: "5-8 mi easy/mod" or
"30-50 min easy." The athlete chooses where in the range based on
how they feel. This is central to the SWAP philosophy: "plans written
in pencil."

## 7. Steady Runs (50K Effort)

**Definition:** Sustained moderate-to-hard running at approximately 50K
race effort. Harder than easy/moderate, easier than threshold.

**Execution:**
- Warm up 15-20 min easy
- Settle into steady effort for prescribed duration
- Cool down 10 min easy
- HR: upper Zone 2 to low Zone 3
- RPE: 6-7 out of 10

**Where it appears:**
- Post-interval work ("2mi steady after the reps")
- Standalone mid-week run in ultra plans
- Long run structured segments ("20min steady within long run")

**Purpose:** Lactate shuttling, fat oxidation at higher output,
race-specific fitness for 50K+ distances.

---

## 8. Power Hour (60-90 min Mod/Hard)

**Definition:** A sustained hard effort of 60-90 min at moderate-to-hard
pace, typically on terrain. The signature SWAP ultra workout.

**Execution:**
- Warm up 20-30 min easy
- Settle into moderate effort
- Progress naturally to mod/hard over 60-90 min
- Cool down 10-20 min easy
- Total run: 90-120 min

**Placement:** Appears 2-3 times in a 12-16 week plan, always on
Saturday. Typically in weeks 5, 8, 14 of champion plans.

**Purpose:** Simulates the sustained output of ultra racing. Tests
fueling, mental fortitude, and physiological resilience simultaneously.

---

## 9. Fatigue Resistance Hills

**Definition:** Short hills at the END of long runs. Not the same as
standalone hill intervals.

**Execution:**
- Run long (12-20 mi easy/mod)
- In final 2-3 miles, do 4-8×30-45s hills at 3K-5K effort
- Jog down recovery
- Cool down easy

**Purpose:** Trains the neuromuscular system to produce force when
glycogen-depleted and mechanically fatigued. This is the direct
training stimulus for physiological resilience.

**Key distinction:** These are NOT hill intervals. They are fatigue
resistance work that happens to use hills as the stimulus. The fatigue
from the long run is the primary training variable.

---

## 10. Back-to-Back Long Days

**Definition:** Saturday long run followed by Sunday long-ish run.
Ultra-specific.

**Examples from plans:**
- Sat: 16-20 mi with structured elements → Sun: 8-12 mi easy w/hills
- Sat: 20-25 mi easy → Sun: 5-10 mi easy/mod
- "Peak weekend" variant: Sat: 20+ mi → Sun: 10-14 mi

**Purpose:** Accumulated fatigue over consecutive days simulates ultra
racing without the single-session volume that risks injury. The Sunday
run starts on "pre-fatigued" legs.

**Recovery:** Monday is always complete rest or easy x-train after
back-to-back weekends.

## 11. Structured Long Runs (3 Types)

All SWAP plans use three rotating long run structures:

### Type A: Easy/Mod Progressive
- Start easy, progress naturally to moderate in final third
- No structured segments
- "Just run and see what happens"

### Type B: Threshold/Tempo Segments
- 15-20 min easy warm-up
- 1-3 structured segments at threshold or marathon effort
- Examples: 3×1mi @ 1hr effort, 20min @ 1hr effort
- 10 min easy cool-down

### Type C: Fatigue Resistance
- Easy/mod for most of the run
- Final 2-3 miles: hills, strides, or pace pickups
- Examples: 5×30s hills, 4×2min @ 10K effort
- Tests output when tired

**Rotation:** A → B → C → A → B → C. Each week gets a different type.
This prevents staleness and ensures all three adaptation targets (pure
aerobic, sustained speed, fatigue resistance) get trained.

## 12. Workout Combinations

Several SWAP workouts combine elements:

### Short Hills + Steady ("The Combo")
- 8-12×30-45s hills at 3K effort
- Recovery: jog down
- Then: 1-3 mi steady at 50K effort
- Purpose: power development → lactate shuttling practice

### Fast Intervals + Threshold
- 6-10×1min fast at 5K effort / 90s easy
- Then: 1-2×10min at threshold (1hr effort)
- Purpose: open the VO2 ceiling, then practice sustaining it

### Track + Moderate Finish
- 2-3 sets of track intervals (400-800)
- Then: 1-2 mi at moderate effort
- Purpose: speed → sustained aerobic output

## Implications for Generator

1. **Workout `segments` schema must support all combos:** hills+steady,
   speed+threshold, track+moderate. These are single workouts with
   multiple segment types.
2. **Strides require a 3-week onramp** for new athletes. The generator
   should check training history before prescribing full stride volume.
3. **Hill grade matters.** The `segments` schema should include `grade`
   field for hill intervals and uphill TM.
4. **Three long-run types must rotate.** The generator should alternate
   A/B/C structures, not repeat the same pattern.
5. **Easy/mod descriptions should use sensory language,** not pace:
   "Start easy, let the run come to you."
6. **Fatigue resistance hills are NOT hill intervals.** They require
   prior long-run fatigue as the setup. Generator should place them
   at the end of long runs, not as standalone sessions.
7. **Power Hour is a specific workout type** that should appear in
   ultra plans — sustained 60-90 min at mod/hard effort. Not "just
   a long run."
8. **Back-to-back weekends** should be explicitly scheduled in 50K+
   plans, with Monday rest enforced.
